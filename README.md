<h1>🎓 UNES Universidade</h1>

<p>
Projeto de uma universidade fictícia desenvolvido como parte dos meus estudos em <strong>HTML5</strong>.
</p>

<h2>📚 Sobre o Projeto</h2>

<p>
A <strong>UNES Universidade</strong> é um site institucional fictício criado com o objetivo de colocar em prática os conhecimentos adquiridos em HTML5.
</p>

<p>
Este projeto faz parte do meu processo de aprendizado em desenvolvimento web e tem como foco a construção de páginas estruturadas, organizadas e semânticas utilizando apenas HTML5.
</p>

<h2>🚀 Objetivo</h2>

<ul>
    <li>Praticar a estruturação de páginas web com HTML5</li>
    <li>Aplicar conceitos de semântica</li>
    <li>Criar um site institucional simples e funcional</li>
    <li>Consolidar os fundamentos do desenvolvimento web</li>
</ul>

<h2>🛠️ Tecnologias Utilizadas</h2>

<ul>
    <li>HTML5</li>
</ul>
